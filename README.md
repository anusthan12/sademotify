# Sad Emotify — Sad Playlist Player

The sad emotion playlist player built for the **Emotify** project — a React-based player that serves music matched to a sad emotional state.

[![React](https://img.shields.io/badge/React-Player_UI-20232a?style=flat-square&logo=react&logoColor=61DAFB)](#)
[![Vercel](https://img.shields.io/badge/Live-sademotify.vercel.app-black?style=flat-square&logo=vercel)](https://sademotify.vercel.app)
[![MIT License](https://img.shields.io/badge/License-MIT-green?style=flat-square)](LICENSE)

## Overview

`sademotify` is the sad-emotion variant player of the Emotify project — a standalone React UI that plays a curated sad playlist when the emotion-detection model returns a sad result.

**Live Player:** [sademotify.vercel.app](https://sademotify.vercel.app)

## Project Structure

```
sademotify/
├── src/          # React player components and logic
├── public/       # Static assets
├── package.json  # Dependencies
└── .gitignore
```

## Running Locally

```bash
npm install
npm start
```
